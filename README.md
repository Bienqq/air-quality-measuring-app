# Pollution Viewer
| Vue | Vuetify |
|--|--|

**Pollution Viewer** - application to measure quality of air in most cities located some countries

##
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
